
# 说明

新型表格界面可以用在文件列表展示等场景。以下是新型表格界面和普通列表界面的区别


|   | 新型表格 | 普通表格或者普通列表 |
| --- | --- | --- |
| 列类型 | 支持多种列类型，不同列类型有相应的格式化显示的规范 | 没有列类型的概念 |
| 横向宽度 | 固定宽度，超出界面显示区域时，内部显示滚动条 | 自适应宽度，可以设置最小宽度。大于最小宽度时，不显示滚动条。 |
| 列宽度 | 用户可以调整宽度 | 用户不能调整每个列的宽度。列宽度分为固定宽度和自适应宽度两种。所有列加起来的宽度是自适应的(见上文)。 |
| 列顺序 | 用户可以调整列的顺序 | 用户不能调整列的顺序 |
| 首列固定 | 提供首列滚动时固定的功能 |   |
| 显示行数 | 采用滚动窗口技术，可以显示数万条记录。一般限制为 5 万条。 | 不采用滚动窗口技术，一般实现为下拉显示更多。或者分页显示，一般一页显示 100 条。 |
| 分页 | 一般不提供分页，最多显示 5 万条，更多数据建议用户使用过滤功能来筛选查看。 | 支持分页。一般一页显示 100 条。 |
| 搜索 | 支持搜索 | 不提供搜索 |
| 排序 | 支持 | 支持 |
| 分组 | 提供分组功能，分组在前端实现。 | 不实现分组功能。 |
| 列隐藏 | 支持用户定制要显示的列 | 一般不支持用户定制要显示的列 |
| 行展开 | 一般提供行展开功能， ﻿在对话框中或者右侧边栏显示行的详情。 | 一般不提供行展开 |
| 标题列点击 | 点击标题列进入项目页 (比如文件页面) | 点击标题列进入项目页 |
| 单元格编辑 | 提供单元格编辑的功能，对不同列类型提供不同的编辑控件。 | 不提供单元格编辑的功能。 |
| 单元格多选 | 提供单元格多选和批量复制、粘贴的功能 |   |
| 单元格下拉填充 | 提供单元格下拉填充的操作 |   |
| 单元格复制黏贴操作 | 提供单元格复制黏贴操作 |   |



# 默认列宽度

## 第一组 文本和长文本

文本列和长文本列默认设置为 350px 宽。

## 第二组 单选列

单选列常用于分组。120px 宽。

## 第三组 多选类列

展示问题相关的负责人、参与人（或者增加评审人等）、标签，都属于多选。

* 负责人：支持多个用户，200px
* 协作人：支持多个用户，200px
* Tags 标签：支持多选标签，200px
这三列设置宽度 200px，通常可以展示前2个选择的项目，表示列支持多选。

## 第四组 时间列

时间列默认设置为 200px。比如

* 创建时间：只读，宽度 200px
* 最后修改时间：只读，宽度 200px
