Ubuntu

{% blocktrans %}Ubuntu users can install Seafile client from the Official PPA:{% endblocktrans %}

sudo add-apt-repository ppa:seafile/seafile-client
sudo apt-get update
sudo apt-get install seafile-gui

{% trans "If you only want to install the command-line client, run sudo apt-get install seafile-cli instead." %}

{% trans "If you want to install the debug symbols (for example, when you want to report a bug), you can install the following packages:" %}

sudo apt-get install libsearpc-dbg ccnet-dbg libccnet-dbg seafile-daemon-dbg libseafile-dbg seafile-gui-dbg

{% trans "Now you can start seafile client from Unity's dash." %}

Debian

{% trans "Debian users can install Seafile client from our debian repo:" %}

{% trans "To install the client, first add the signing key:" %}

sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 8756C4F765C9AC3CB6B85D62379CE192D401AB61

{% trans "Then add the repo to your apt source list, using the line corresponding to your debian version:" %}

# For Debian 7
echo deb http://deb.seadrive.org wheezy main | sudo tee /etc/apt/sources.list.d/seafile.list

# For Debian 8
echo deb http://deb.seadrive.org jessie main | sudo tee /etc/apt/sources.list.d/seafile.list

# For Debian 9
echo deb http://deb.seadrive.org stretch main | sudo tee /etc/apt/sources.list.d/seafile.list

{% trans "Update your local apt cache :" %}

sudo apt-get update

{% trans "Now install the client:" %}

sudo apt-get install seafile-gui

{% trans "If you only want to install the command-line client, run sudo apt-get install seafile-cli instead." %}

{% trans "Centos/RHEL (community maintained)" %}

{% trans "Since 7.0.3 version, we provide official repo for CentOS or RHEL. Currently only CentOS/RHEL 7 is supported." %}

{% trans "Add the repo (The same repo is used for seadrive.)" %}

wget -O- https://git.io/seadrive-centos7-repo | sudo tee /etc/yum.repos.d/seadrive.repo

{% trans "Install Seafile Client" %}

sudo yum install -y epel-release
sudo yum install -y seafile --enablerepo=cr

{% trans "Fedora (community maintained)" %}

{% blocktrans %}There is a community maintained Seafile Client RPM package in Fedora's official repository.{% endblocktrans %}
